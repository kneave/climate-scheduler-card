# Climate Scheduler Card

Full-featured scheduling interface for your climate entities.

## Requirements

This card requires the **Climate Scheduler integration** to be installed separately.

Install the integration from HACS or [GitHub](https://github.com/kneave/climate-scheduler).

## Quick Start

1. Add the card to your dashboard
2. Select a climate entity or create groups
3. Drag nodes on the graph to create your schedule
4. Enable the schedule

## Features

- Interactive graph editor with drag & drop
- Multiple schedule modes (all days, 5/2, individual days)
- Temperature history visualization
- Group management for multiple thermostats
- Full HVAC control support
- Mobile-friendly interface
